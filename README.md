# Study Abroad App

A React Native mobile application built with Expo.

## Features

- List of study abroad universities
- University details page
- Responsive UI
- React Navigation
- Reusable Program Card component

## Installation

1. Clone repository

git clone <repository-url>

2. Install dependencies

npm install

3. Run app

npx expo start

## Technologies

- React Native
- Expo
- React Navigation
